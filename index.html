CSS Selectors

Selector Description Example
Element Selects HTML elements based on the element name h2 {font-size: 30px; font-family: "Times New Roman"}
Class Selects HTML elements based on their "class" (assigned in the opening HTML tag) .green {color: green}
ID Selects a single HTML element based on its "id" (assigned in the opening HTML tag) #paradise {background-image: url("./Resources/Images/paradise.jpeg");}

Common CSS Properties

Property Description Example
font-family Changes the default typeface of the selected element; assign multiple values to this property to ensure a web safe font is available font-family: Helvetica, Times, "sans serif"
font-size Changes the default size of the text font-size: 100px; font-size: 2.5em
color Changes the default color of the selected element color: rgb(255, 0 , 0); color: hsla(35, 100%, 50%, .5)
background-color Sets the background color of the selected element background-color: seagreen; background-color: #9F773A
position Changes the position of the element; values: static, relative, absolute, fixed, or sticky position: sticky; position: relative
